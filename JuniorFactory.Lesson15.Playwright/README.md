# 15. Playwright

## Краткое описание

То же, что Selenium из прошлого урока, только на Playwright – современнее и шустрее. Гоняем браузер из консольного
приложения. Лежит со своим `.sln` в папке `PlaywrightConsoleApp/`.

## Что нужно

Перед первым запуском поставь браузеры:

```bash
pwsh bin/Debug/net8.0/playwright.ps1 install
```
