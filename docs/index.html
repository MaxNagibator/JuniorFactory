Первая смена

Три шага, чтобы развернуть цех у себя и запустить первый урок. Дальше – по конвейеру.

  1. 01

    Поставь инструменты

    Скачай бесплатный .NET SDK 8 и любую IDE: Visual Studio, Rider или VS Code.

  2. 02

    Склонируй цех

    Забери весь код курса одной командой:

    git clone https://github.com/MaxNagibator/JuniorFactory.git
  3. 03

    Запусти первый урок

    Открой JuniorFactory.sln в IDE, выбери проект Lesson01 и нажми «Запустить». Видео и конспект – на странице урока 01.
