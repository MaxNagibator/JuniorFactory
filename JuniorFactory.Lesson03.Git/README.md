# 3. Основы Git

## Краткое описание

Познакомились с основными функциями git.

SSH ключ на коленке к githubу добавили.

## Что нужно

- Установленный **Git** (по желанию GUI вроде TortoiseGit) и аккаунт **GitHub** для SSH-ключа.
- **.NET 8 SDK** – если хочешь погонять мини-проект из урока.
