# 11. Асинхронность и параллелизм

## Краткое описание

- обзор параллельной обработки задач синхронного кода
- разбор асинхронности:
  как запустить задачки в параллель
  как отменить задачу
- всех с днём знаний ^_^

- **.NET 8 SDK** – консольные примеры с `Task`, `AsParallel` и отменой задач.
