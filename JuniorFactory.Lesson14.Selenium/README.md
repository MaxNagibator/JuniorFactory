# 14. Selenium

## Краткое описание

Подёргали Selenium: учим консольное приложение открывать сайт, искать элементы на странице, кликать и проверять
результат. Лежит со своим `.sln` в папке `SeleniumConsoleApp/`, так что можно открыть отдельно.
